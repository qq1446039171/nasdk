# 《纳斯达克 60% 严格执行方案 v1.2》

## 一、策略定位

本策略用于管理个人账户中纳斯达克相关资产的长期配置与买入节奏。  
本策略不预测短期涨跌，不进行短线交易，不因情绪临时改变规则。  
本策略的目标是：在可控风险下，持续 10 年以上参与纳斯达克长期增长。

## 二、最高规则

1. 纳指相关总敞口上限为 `60%`。
2. 其中：
   - `40%` 为已买入纳指资产上限
   - `20%` 为纳指备用资金上限
3. 任意时点：
   - 主动买入不得使“已买入纳指资产”超过 `40%`
   - 主动买入不得使“纳指相关总敞口”超过 `60%`
4. 超过 `40%` 的已买入比例，只允许由以下两种情况被动形成：
   - 回撤触发后的规则内买入
   - 市场上涨导致持仓市值自然上升
5. 备用资金只能用于买入纳指，不得挪作他用，不得提前使用。
6. 本策略不包含短线卖出机制；“停止买入”不等于“卖出”。

## 三、适用范围

1. 纳指标的包括：
   - 场外纳斯达克指数基金，固定定投 `120 元/工作日`
   - 场内 `513100`
2. 纳指相关总敞口定义为：
   - 已买入纳指资产市值
   - 加上纳指备用资金余额
3. 场外定投金额计入“已买入纳指资产”和“纳指相关总敞口”，但不计入“备用资金池”。

## 四、基准参数

1. 初始总资产基准：`190,000`
2. 初始目标金额：
   - 已买入纳指资产目标上限：`76,000`
   - 纳指备用资金目标上限：`38,000`
3. 回撤判定基准：
   - 使用 `纳斯达克100指数（NDX）`
   - 以“最近 1 年高点”为回撤锚点
4. 回撤计算公式：
   - 回撤幅度 = `（最近 1 年高点 - 当前 NDX 点位） / 最近 1 年高点`

## 五、账户隔离原则

1. 本策略中的 `20%` 备用资金是投资备用资金，不是生活应急金。
2. 生活应急金必须独立于本策略之外，不得计入备用资金。
3. 若生活现金流承压，应优先保护生活安全，不得硬执行本策略。

## 六、阶段一：主动建仓规则

目标：将“已买入纳指资产”逐步建立到 `40%` 上限附近。

执行规则：

1. 建仓周期：`6-8 周`
2. 买入频率：每周一次，固定星期执行
3. 每周买入额度：
   - 最低 `5,000`
   - 最高 `7,000`
4. 买入工具：`513100`
5. 若本周已触发回撤买入，则本周不再额外执行主动建仓买入。
6. 若已买入纳指资产达到或超过 `40%`，阶段一自动结束。

补充说明：

1. 若账户当前已持有纳指仓位，则阶段一以“达到 `40%` 上限”为结束条件，不强求满 `6-8 周`。
2. 若从 `0` 开始建仓，而每周上限不足以在 `8` 周内达到目标，则允许延长周数，但仍必须遵守每周额度上限。

## 七、阶段二：备用资金使用规则

备用资金的唯一用途是：在市场回撤时，分档买入纳指。

### 1. 触发条件

仅当 `NDX` 相对最近 `1` 年高点的回撤达到指定档位时触发。  
基金净值、ETF 溢价、汇率波动仅作为执行参考，不作为触发依据。

### 2. 快照机制

1. 当本轮回撤首次达到 `-10%` 时，立即记录并锁死当时的备用资金余额。
2. 该金额定义为“本轮回撤快照备用金”。
3. 本轮 `-10% / -15% / -20% / -25%` 的买入金额，全部按该快照金额计算。
4. 本轮未结束前，不因后续新增现金、场外定投或市场变化修改该快照金额。

### 3. 分档使用比例

1. 回撤达到 `-10%`：使用快照备用金的 `20%`
2. 回撤达到 `-15%`：再使用快照备用金的 `30%`
3. 回撤达到 `-20%`：再使用快照备用金的 `30%`
4. 回撤达到 `-25%`：使用快照备用金剩余全部

### 4. 执行约束

1. 每一档只触发一次。
2. 不重复触发。
3. 不回补已触发档位。
4. 不提前预判。
5. 不允许因“感觉快到底了”提前动用后续档位资金。

### 5. 本轮结束与重新触发规则

1. 一轮回撤自 `NDX` 首次触发 `-10%` 起开始计算。
2. 本轮回撤中，每个档位只允许触发一次。
3. 若某档位已经触发，即使市场随后反弹、再度回落至同一档位，也不得重复触发该档位。
4. 例如：先回撤至 `-13%`，已触发 `-10%` 档；之后即使反弹到 `-3%`，再回撤到 `-10%`，也不得再次使用 `-10%` 档备用金。
5. 只有当本轮回撤正式结束后，下一轮再次回撤至 `-10%`，才允许重新启用 `-10%` 档。

### 6. 本轮回撤结束条件

满足以下任一条件，视为本轮回撤结束：

1. `NDX` 回撤收窄至 `1%` 以内，并连续维持至少 `5` 个交易日
2. `NDX` 创出新的近 `1` 年高点

### 7. 新一轮回撤规则

1. 只有在上一轮回撤结束后，下一次再度触发 `-10%`，才建立新的回撤轮次。
2. 新回撤轮次应重新建立快照备用金，并重新计算四档金额。

## 八、场外定投规则

1. 固定执行：`120 元/工作日`
2. 不因涨跌暂停，不因情绪调整，不因媒体观点中断
3. 场外定投视为长期底仓建设的一部分
4. 场外定投计入已买入纳指资产，但不计入备用资金使用额度

## 九、自动停止与冻结规则

满足任一条件，立即停止所有主动买入，仅保留场外 `120 元/工作日` 定投。

触发条件：

1. 已买入纳指资产 `>= 40%`
2. 纳指相关总敞口 `>= 60%`
3. 连续 `3` 周上涨，且本人明确出现“怕踏空、想追买”的情绪

冻结状态下：

1. 停止主动建仓
2. 停止非规则内临时加仓
3. 保留场外固定定投
4. 不允许用主观判断解除冻结

## 十、备用资金回补规则

1. 当备用资金因回撤执行而被部分或全部使用后，不得通过主观判断立即重建。
2. 备用资金的默认回补方式为：后续新增现金流优先补回备用资金池。
3. 在备用资金未补足至目标上限前：
   - 停止新的主动建仓买入
   - 保留既定场外 `120 元/工作日` 定投
   - 不允许额外扩大纳指相关总敞口上限
4. 不以“上涨后卖出一部分持仓补回备用金”作为默认机制。
5. 只有在季度复核时，若纳指相关敞口因上涨已显著超出既定风险边界，才允许通过组合再平衡卖出部分纳指资产，将资金补回备用资金池。
6. 再平衡卖出的目的仅为恢复风险结构与备用资金缓冲，不得作为短线择时或主动止盈行为。
7. 备用资金补足前，所有新增非定投资金原则上优先进入备用资金池，而非新增主动买入。

## 十一、极端行情规则

1. 若市场跌破 `-25%` 后继续下跌：
   - 不新增临时备用金
   - 不扩大纳指上限
   - 不新增档位
   - 仅保留既定仓位和场外 `120 元/工作日` 定投
2. 不得因为“跌得太多看起来很便宜”而突破 `60%` 上限。

## 十二、季度复核规则

每 `3` 个月允许进行一次复核。  
复核是唯一允许调整金额基准的时间窗口。

仅允许做的事情：

1. 重新计算总资产基数
2. 重算 `40%` 与 `20%` 对应金额
3. 更新账户当前已买入金额、备用金额和敞口比例

不允许做的事情：

1. 因短期涨跌修改 `40% / 20%` 比例
2. 临时改变回撤档位
3. 临时改变每档触发比例
4. 因情绪调整策略核心框架

## 十三、再平衡原则

1. 本策略不做短线择时卖出。
2. 若因价格上涨导致纳指资产占比显著高于原定风险承受区间，可在季度复核时评估是否需要组合级再平衡。
3. 再平衡的目的仅是恢复风险边界，不是做波段获利。
4. 若暂无完整组合级卖出规则，则默认“不主动卖出，只停止新增主动买入”。

## 十四、执行记录要求

每次执行都必须记录：

1. 日期
2. 执行动作
3. 执行金额
4. 触发原因
5. 当前已买入比例
6. 当前总敞口比例
7. 当前是否冻结

建议记录四类动作：

1. 主动建仓买入
2. 回撤档位买入
3. 场外定投
4. 冻结 / 解除冻结

## 十五、纪律声明

1. 本策略优先级高于情绪。
2. 本策略优先级高于短期新闻。
3. 本策略优先级高于临时预测。
4. 一切例外只能在季度复核中讨论，不能在盘中决定。

## 十六、策略信条

我不是在押注纳斯达克的短期涨跌，而是在管理一场可以持续 10 年的高仓位战役。
